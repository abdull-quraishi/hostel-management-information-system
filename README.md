# 🏨 Hostel Management Information System (HMIS)

A web-based Hostel Management Information System developed to simplify and automate hostel operations. The system helps administrators manage students, rooms, bookings, payments, and daily hostel activities through an organized dashboard.

## ✨ Features

- 🔐 User Authentication
- 📊 Admin Dashboard
- 👨‍🎓 Student Management
- 🛏 Room Management
- 📝 Hostel Registration Management
- 💰 Payment Management
- 📋 Booking Records
- 📱 Responsive User Interface


## 🛠 Technologies Used

- PHP
- Laravel
- MySQL
- HTML5
- CSS3
- Bootstrap
- JavaScript
- Git & GitHub


## 📂 System Modules

### Admin Panel
- Manage students
- Manage rooms
- View hostel records
- Manage payments
- Monitor hostel activities

### Student Management
- Student information management
- Hostel allocation records
- Registration tracking
